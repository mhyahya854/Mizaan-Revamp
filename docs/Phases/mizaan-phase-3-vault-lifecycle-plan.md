# Phase 3 Vault Lifecycle Plan

Reference: `Mizaan Ultimate Plan.txt`.

## Goal

Add a vault lifecycle boundary for the current prototype local provider.

## Scope

- Current vault session model.
- Recent vaults list.
- Demo/prototype local vault mode representation.
- Provider capabilities.
- Vault health.
- `/vault` route.
- Sidebar and Settings links.

## Intentional Limits

- No real folder picker.
- No real SQLite.
- No real lock file.
- No portable USB vault readiness claim.

## Status

Phase 3 - partially implemented — current session, recent vaults, prototype local mode, provider capabilities, vault health, `/vault`, sidebar link, Settings link, and command palette route command exist. Demo mode is represented by the session type but does not yet have an isolated demo storage sandbox.
